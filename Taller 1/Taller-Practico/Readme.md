Aplicación web desarrollada con TypeScript y Vite que consume la [Dragonball API](https://web.dragonball-api.com) para explorar información sobre personajes, planetas y transformaciones del universo Dragon Ball.

## ✨ Características

- **Búsqueda inteligente**: Permite buscar por nombre o ID en tres categorías:
  - Personajes
  - Planetas
  - Transformaciones
- **Visualización detallada**: Muestra información relevante como nivel de ki, raza, género, afiliación y descripción
- **Diseño responsive**: Interfaz adaptable a dispositivos móviles, tablets y desktop
- **Manejo de errores**: Sistema robusto con reintentos automáticos y mensajes amigables
- **Carga optimizada**: Loader animado y cancelación de peticiones en curso

## 🛠 Tecnologías Utilizadas

- **TypeScript**: Tipado estático y programación orientada a objetos
- **Vite**: Build tool y servidor de desarrollo rápido
- **HTML5**: Estructura semántica de la aplicación
- **CSS3**: Estilos personalizados con variables CSS y diseño responsive
- **Dragonball API**: Fuente de datos principal


## 🔧 Instalación y Uso

### Prerrequisitos
- Node.js (v14 o superior)
- npm o yarn

### Pasos de instalación

1. **Clonar el repositorio**
   ```bash
   git clone https://github.com/tu-usuario/Taller1-Practico.git
   cd Taller1-Practico
2. **Instalar dependencias**
   npm install
3. **Iniciar servidor de desarrollo**
   npm run dev

4. **Abrir la aplicación**
   Navega a http://localhost:5173/ en tu navegador
